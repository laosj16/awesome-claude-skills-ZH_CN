# 项目描述

## Awesome Claude Skills 中文版

本项目是对原始 [Awesome Claude Skills](https://github.com/anthropics/skills) 项目的完整汉化版本，旨在为中文用户提供更友好和便捷的使用体验。

### 什么是本项目

这是一份精心整理的 **1000+ 个生产可用的 Claude Skills 与插件合集**，涵盖文档处理、开发工具、数据分析、商业营销、通信写作、创意媒体、生产力工具、项目协作、安全系统等多个领域。

原项目由 Anthropic 和 Composio 社区维护，我们对其进行了完整的中文翻译和本地化调整，使中文开发者和用户能够：

- 🎯 快速理解每个 Skill 的功能和应用场景
- 📚 获得清晰的中文说明和使用指南
- 🔗 轻松在中英文版本间切换
- 💡 发现适合自己工作流程的工具和自动化方案

### 核心内容

- **1000+ 个 Claude Skills** — 从文档处理到 API 集成，覆盖各类应用场景
- **78 个 SaaS 应用自动化** — 通过 Composio 连接 Gmail、Slack、GitHub、Notion 等常用工具
- **双语支持** — 中文版（README.md）和英文版（README_us.md）无缝切换
- **快速上手教程** — 在 Claude.ai、Claude Code 和 API 中使用 Skill 的完整指南
- **社区资源** — 官方文档链接、创建 Skill 的最佳实践、贡献指南

### 为什么选择本项目

✨ **完整汉化** — 不只是翻译工具列表，而是全面汉化的知识库  
🚀 **易于上手** — 中文说明让开发者快速理解和应用  
🔄 **保持更新** — 跟踪原项目更新，定期同步最新的 Skill  
🤝 **社区驱动** — 欢迎贡献改进和新的 Skill 推荐  

### 快速开始

1. 浏览 [README.md](./README.md) — 完整的中文技能列表
2. 在 Claude.ai、Claude Code 或通过 API 使用感兴趣的 Skill
3. 参考[创建技能](./README.md#创建技能)章节开发自己的 Skill
4. 在[参与贡献](./README.md#参与贡献)章节提交你的想法和改进

### 致谢

感谢 Anthropic、Composio 和所有 Skill 贡献者为 Claude 生态的繁荣做出的贡献。本项目旨在让更多中文用户能够受益于这个强大的工具集。
